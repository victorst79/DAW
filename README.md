# Exercise 1 - Calculator Bash

Implement a script that shows the following menu:

1 - Add

2 - Subtract

3 - Divide

4 - Multiply

0 - Exit
 
• After displaying the menu, you will be asked to choose an option. If the chosen option is not between 1 and 4, an error message will be displayed. In case the option is valid, two numbers will be requested per keyboard and depending on the operation chosen, the result will be returned per screen.

• Each operation will be implemented using functions. Each function header must precede information about it: Name of the function, arguments it receives and description. Use a template like this:

    #######################
    # Function:
    # Arguments:
    # Description:
    #######################

• If the chosen option is not valid, an error message will be indicated to the user and the menu will be displayed again.

• The program will end, when 0 is pressed.

• Include at the beginning of the script information of the student who delivers the practice:

- Name and surname

-  Course

• Considerations:

- Make use of comments in the code.
- Optimized use of variables.
- Code ordered and readable.
- Do not repeat code unnecessarily.
- Error control.
